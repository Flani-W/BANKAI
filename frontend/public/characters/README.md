# Character portraits (optional drop-in)

Put a portrait here named after the character id to use real art instead of the
procedural placeholder:

```
ichigo.jpg  byakuya.jpg  toshiro.jpg  rukia.jpg
renji.jpg   kenpachi.jpg mayuri.jpg   soifon.jpg
```

- Format: `.jpg` (the loader looks for `<id>.jpg`). Recommended ~800×1067 (3:4).
- If a file is missing, the app renders a procedural portrait automatically — nothing breaks.
- Use art you have the rights to. Don't commit copyrighted images to a public repo.
